# Panel Elecciones Andalucía 2026

Dashboard para seguir en directo los resultados de las elecciones al Parlamento de Andalucía del 17 de mayo de 2026.

## Qué hace

- Muestra el porcentaje de escrutinio en tiempo real
- Distribuye los escaños en un hemiciclo SVG
- Visualiza la participación por provincia y municipio
- Embebe mapas y gráficos de Datawrapper actualizados automáticamente

## Stack

React 18 + Vite 5 · Tailwind CSS · react-router-dom · GitHub Pages
